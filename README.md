### Degit Conveter

 - This code is a decimal converter that converts a decimal number to a number based on the number received.

 - This code is written in assembly language pep8.
 
### 진수 변환기

 - 어셈블리어로 작성된 진수 변환기입니다.
 
 - 2진수, 8진수, 16진수로 변환 가능합니다.
